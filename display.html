display

display применятся к родительскому элементу (Flex-контейнер). От переданного значения зависит каким будет flex-контейнер (блочным или строчным).

Flex-контейнер не является блочным контейнером, поэтому для дочерних элементов не работают такие CSS-свойства, как float, clear, vertical-align.

.flexContainer {

display: flex;// блочный уровень

}

.flexContainer {

display: inline-flex;// строковый уровень

}
